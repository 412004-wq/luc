## 🎯 使用教學

### 場景 1：用外幣付款（如美元）
```
1. 消費項目：輸入「機票」
2. 外幣幣別：搜索「USD」→ 選擇「美元 (USD)」
3. 金額：輸入「1000」
4. 點擊「🔄 自動獲取匯率換算」
5. 系統自動填入：匯率 30.5000，台幣 30,500.00
6. 點擊「💾 儲存消費」✓
```

### 場景 2：用台幣付款
```
1. 消費項目：輸入「咖啡」
2. 外幣幣別：搜索「TWD」→ 選擇「台幣 (TWD)」
3. 金額：輸入「100」
4. 自動填入：匯率 1，台幣 100
5. 點擊「💾 儲存消費」✓
（注意：此時「自動獲取匯率」按鈕隱藏）
```

### 場景 3：搜索新幣別（如人民幣）
```
1. 外幣幣別搜索欄：輸入「CNY」
2. 下拉菜單出現選項
3. 點擊「人民幣 (CNY)」
4. 系統記住選擇，輸入金額
5. 點擊「🔄 自動獲取匯率換算」
6. 自動計算並填入台幣金額
```
